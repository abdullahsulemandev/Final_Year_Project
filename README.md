 Video Analyzer :

A simple flask web app that performs  Video Analyzer in video using YOLOv8 ultralytics. 



#### Features
- Video uploading
- Live video feed
- Real-time object detection and bounding boxes
- Login and Signup
- Detected Video Downloading



#### Requirements
- opencv
- ultralytics
- torch
- flask
- reportlab
- pandas
- numpy
- YOLOv8
- datetime
- url_for
- redirect
- render_template
- request
- session
- os
- time
- werkzeug.security
- flash
